C. Sifat - Sifat Bilangan Bulat

Ada 3 macam sifat - sifat dalam bilangan bulat, yaitu:

a. Sifat Komutatif (Pengelomp[okan)

Sifat komutatif bisa menukarkan bilangan yang tadinya berada disebelah kanan bisa ditukarkan menjadi di sebelah kiri. Sifat ini hanya ada pada 2 jenis operasi bilangan bulat, yaitu : penjumlahan dan perkalian

Sifat ini tidak akan merubah hasilnya jika ditukarkan lalu dijumlahkan atau dikalikan akan tetap sama hasilnya. Sifat ini pula hanya berlaku untuk penjumlahan bilangan bulat positif dengan bilangan bulat positif dan perkalian saja.

- Pada Penjumlahan

a + b = b + a

- Pada Perkalian

a x b = b x a

Contoh:

1. 5 + 6 = 6 + 5

2. 6 x 7 = 7 x 6

b. Sifat Asosiatif (Penyebaran)

Sifat asosiatif merupakan pengelompokan antara tiga bilangan. Sifat ini juga berlaku 2 jenis operasi bilangan, yaitu : penjumlahan dan perkalian. Pengelompokan ini akan tetap sama hasilnya.

- Pada Penjumlahan

a + (b + c) = (a + b) + c

- Pada Perkalian

a x (b x c) = (a x b) x c

Contoh:

2. 4 + (6 + 8) = (4 + 6) + 8

3. 2 x (6 x 5) = (2 x 6) x 5

c. Sifat Distributif (Penyebaran)

Sifat distributif merupakan penyebaran antara tiga bilangan dengan 2 operasi perhitungan. Sifat ini sangat erat dengan operasi bilangan. Dimana bentuk dari sifat ini digunakan dalam bentuk penjumlahan dan pengurangan yang disertai perkalian yang ada dalam bentuk ini.

- Pada Penjumlahan

a x (b + c) = (a x b) + (a x c)

- Pada Pengurangan

a x (b – c) = (a x b) – (a x c)

Contoh :

2. 4 x (9 + 6) = (4 x 9) + (4 x 6)

3. 3 x (5 – 6) = (3 x 5) – (3 x 6)
