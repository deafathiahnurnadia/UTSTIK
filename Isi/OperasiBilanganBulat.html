B. Operasi Bilangan Bulat

a. Penjumlahan Bilangan Bulat

Apabila menggunakan garis bilangan, menjumlahkan dengan bilangan bulat positif maka banyaknya langkah ke kanan dan apabila menjumlahkan dengan bilangan bulat negatif maka banyaknya langkah ke kiri.

Bentuknya :

1. a + b = (+)

Penjumlahan bilangan bulat positif dengan bilangan bulat positif akan menghasilkan bilangan bulat positif.

2. –a + (–b) = (-)

Penjumlahan bilangan bulat negatif dengan bilangan bulat negatif akan menghasilkan bilangan bulat negatif.

3. –a + b = (-)

Penjumlahan bilangan bulat negatif dengan bilangan bulat positif akan menghasilkan bilangan bulat negatif.

4. a + (-b) = (-)

Penjumlahan bilangan bulat negatif dengan bilangan bulat positif akan menghasilkan bilangan bulat negatif.

Contoh :

1.2 + 4 = 6

2.-5 + (-3) = -8

3.-4 + 5 = 1

4.3 + (-7) = -4

b. Pengurangan Bilangan Bulat

Apabila menggunakan garis bilangan, mengurangi dengan bilangan bulat positif maka banyaknya langkah ke kiri dan apabila mengurangi dengan bilangan bulat negatif maka banyaknya langkah ke kanan.

Bentuknya:

1.a – b = (+)

Pengurangan bilangan bulat positif dengan bilangan bulat positif menghasilkan bilangan positif. Ini akan berlaku, Jika bilangan bulat positif yang berada di sebelah kiri lebih besar dari bilangan bulat positif yang berada di sebelah kanan.

2.–a – (b) = (-)

Pengurangan bilangan bulat negatif dengan bilangan bulat positif menghasilkan bilangan negatif.

3.a – (-b) = (+)

Pengurangan bilangan bulat positif dengan bilangan bulat negatif menghasilkan bilangan positif.

Contoh :

1.5 – 3 = 2

2.6 – 7 = -1

3.-6 – (2) = -8

4.5 – (-2) = 7

c. Perkalian Bilangan Bulat

Beberapa bentuk pola perkalian bilangan adalah sebagai berikut:

1.a x b = (+)

Perkalian bilangan bulat positif dengan bilangan bulat positif menghasilkan bilangan positif

2.a x (-b) = (-) / (-a) x b = (-)

Perkalian bilangan bulat positif dengan bilangan bulat negatif menghasilkan bilangan negative dan sebaliknya.

3.(-a) x (-b) = (+)

Perkalian bilangan bulat negatif dengan bilangan bulat negatif menghasilkan bilangan positif.

contoh :

1. 2 x 4 = 8

2. 3 x (-5) = -15

3. (-6) x 2 = -12

4. (-4) x (-5) = 20

d. Pembagian Bilangan Bulat

Beberapa bentuk pola pembagian bilangan adalaah sebagai berikut:

1. Pembagian bilangan bulat positif dengan bilangan bulat positif menghasilkan bilangan positif.

2. Pembagian bilangan bulat positif dengan bilangan bulata negatif menghasilkan bilangan negatif dan sebaliknya

3. Pembagian bilangan bulat negatif dengan bilangan bulat negatif menghasilkan bilangan positif.

contoh :

1. 4/2 = 2

2. (-9)/3 = -3

3. (-12)/(-6) =2
